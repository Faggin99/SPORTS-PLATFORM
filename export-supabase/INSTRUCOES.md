# Exportacao de Dados do Supabase

Execute cada query abaixo no **SQL Editor** do Supabase (https://supabase.com/dashboard/project/kashktmwcktxfdjkxgki/sql/new)
e salve o resultado como CSV ou copie o JSON gerado.

## Ordem de execucao:

1. `01_export_users.sql` - Usuarios (MAIS IMPORTANTE)
2. `02_export_all_data.sql` - Todas as tabelas de dados

Apos exportar, salve os JSONs gerados na pasta `export-supabase/data/`
